# Solve File Sample

Solve puzzles in file

## Usage

Run with:
```
node run <filename(s)>
```
Each line in the file represents a game position.

Example:
```
node run problems01.txt
```

## Files
Taken from [Guenter Stertenbrink's Sudoku page](http://magictour.free.fr/sudoku.htm)
